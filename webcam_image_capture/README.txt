# THIS PROGRAMME WILL HELP YOU TO CAPTURE IMAGES FROM A WEBCAM SO YOU CAN BUILD YOUR OWN DATASET FOR TRAINING

1. DO NOT DELETE ANY PARAMETERS IN CONFIGURATION FILE, IF STUFF IS IN QUOTATION MARKS OR PARANTHESIS, LEAVE THEM THERE
2. ENSURE THE CONFIGURATION FILE IS IN THE SAME DIRECTORY AS DETECTION AND TRAINING FILES 
3. YOU CAN CHANGE THE PATHS OF THE FILES AS YOU PLEASE, BUT ENSURE YOU UPDATE IT IN THE CONFIGURATION FILES
4. IF YOU DONT KNOW WHAT YOURE DOING, LEAVE THE CONFIGURATION FILE ALONE
